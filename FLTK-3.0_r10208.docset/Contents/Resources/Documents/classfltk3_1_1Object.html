Detailed Description

A base class for any kind of langage wrapper around FLTK3.

fltk3::Object is a way to inject code into the existing library at run-time. When used as a base class, external languages and interfaces can use the counterpart fltk3::Wrapper to allow for external overrides of every fltk3 class.

Practical examples for this are the FLTK1 and FLTK2 emulation. It all looks a bit convoluted at first, but the use is straight forward and creates a true relationship between this library and any other frontend.
