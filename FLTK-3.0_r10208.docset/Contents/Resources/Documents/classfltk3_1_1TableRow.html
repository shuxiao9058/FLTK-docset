Detailed Description

A table with row selection capabilities.

This class implements a simple table with the ability to select rows. This widget is similar to an fltk3::Browser with columns. Most methods of importance will be found in the fltk3::Table widget, such as fltk3::Table::rows() and fltk3::Table::cols().

To be useful it must be subclassed and at minimum the draw_cell() method must be overridden to provide the content of the cells. This widget does not manage the cell's data content; it is up to the parent class's draw_cell() method override to provide this.

Events on the cells and/or headings generate callbacks when they are clicked by the user. You control when events are generated based on the values you supply for fltk3::Table::when().
