Detailed Description

This is a dummy class that allows you to end a fltk3::Group in a constructor list of a class:

class MyClass {
fltk3::Group group;
fltk3::Button button_in_group;
fltk3::End end;
fltk3::Button button_outside_group;
MyClass();
};
MyClass::MyClass() :
group(10,10,100,100),
button_in_group(20,20,60,30),
end(),
button_outside_group(10,120,60,30)
{}
