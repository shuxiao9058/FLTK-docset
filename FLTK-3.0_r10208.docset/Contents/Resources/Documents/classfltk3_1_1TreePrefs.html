Detailed Description

Tree widget's preferences.

fltk3::Tree's Preferences class.

This class manages the fltk3::Tree's defaults. You should probably be using the methods in fltk3::Tree instead of trying to accessing tree's preferences settings directly.

void fltk3::TreePrefs::showcollapse ( int  val)
inline

Set if we should show the collapse icon or not.

If collapse icons are disabled, the user will not be able to interactively collapse items in the tree, unless the application provides some other means via open() and close().

Parameters
[in]val1: shows collapse icons (default),
0: hides collapse icons.
