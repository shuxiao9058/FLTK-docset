Detailed Description

The fltk3::FileIcon class manages icon images that can be used as labels in other widgets and as icons in the FileBrowser widget.

int fltk3::FileIcon::type ( )
inline

Returns the filetype associated with the icon, which can be one of the following:

  • fltk3::FileIcon::ANY, any kind of file.
  • fltk3::FileIcon::PLAIN, plain files.
  • fltk3::FileIcon::FIFO, named pipes.
  • fltk3::FileIcon::DEVICE, character and block devices.
  • fltk3::FileIcon::LINK, symbolic links.
  • fltk3::FileIcon::DIRECTORY, directories.
