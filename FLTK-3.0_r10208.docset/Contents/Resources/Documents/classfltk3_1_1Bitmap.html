Detailed Description

STore and draw single color images.

The fltk3::Bitmap class supports caching and drawing of mono-color (bitmap) images. Images are drawn using the current color.

fltk3::Image * fltk3::Bitmap::copy ( int  W,
int  H 
)
virtual

The copy() method creates a copy of the specified image.

If the width and height are provided, the image is resized to the specified size. The image should be deleted (or in the case of fltk3::SharedImage, released) when you are done with it.

If the image has been cached for display, delete the cache data.

This allows you to change the data used for the image and then redraw it without recreating an image object.
