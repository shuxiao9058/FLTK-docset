Detailed Description

Typedefs defined in <fltk3/run.h> for callback or handler functions passed as function parameters.

FLTK uses callback functions as parameters for some function calls, e.g. to set up global event handlers (fltk3::add_handler()), to add a timeout handler (fltk3::add_timeout()), and many more.

The typedefs defined in this group describe the function parameters used to set up or clear the callback functions and should also be referenced to define the callback function to handle such events in the user's code.

See Also
fltk3::add_handler(), fltk3::add_timeout(), fltk3::repeat_timeout(), fltk3::remove_timeout() and others
